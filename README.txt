# Fahrenheit's Starcraft 1.16.1 Brood War AI Bot


C++ project for the development of an SC AI Bot.

## Bots Features
  - Protoss focused bot.
  - Able to strictly follow certain strategies.
  - Able to adapt its strategies on the fly.
